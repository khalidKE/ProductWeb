# 🛒 Product CRUD Application

A simple and responsive Product Management CRUD (Create, Read, Update, Delete) application built using **HTML**, **CSS**, and **JavaScript**. This app allows users to add, search, update, and delete product entries within a web interface.

---

## 🔗 Live Demo

👉 [Click here to try the app](https://khalidke.github.io/ProductWeb/)

---

## ✨ Features

- ✅ Add new products with name, price, category, and description  
- 🔍 Search products dynamically by name  
- 🔄 Update existing product details  
- ❌ Delete products from the list  
- 📱 Responsive design with clean and user-friendly UI

---

## 🧩 Technologies Used

- **HTML5** – For structuring the app layout  
- **CSS3** – For styling and responsive design  
- **JavaScript** – For implementing all dynamic and CRUD functionalities

## 🛠️ How It Works

- **Add Product**: Fill in the product fields and click **"add product"**  
- **Update Product**: Click **"Update"** next to a product, edit fields, then click **"update product"**  
- **Delete Product**: Click the **"Delete"** button to remove a product  
- **Search**: Start typing in the search input to filter the list of products by name
